Mythic Bastionland Realms

Procedural Hex Map Generator

Generate procedural hex maps for your Mythic Bastionland adventures. Each hex contains unique terrain, encounters, and points of interest that bring the weird and wonderful world to life.

About the Generator

This hex map generator creates random wilderness regions for Mythic Bastionland using procedural generation algorithms. Each hex is populated with:

  • Terrain Types: Forests, hills, swamps, plains, and more
  • Encounters: Strange creatures and mysterious travelers from the Realms
  • Points of Interest: Ruins, settlements, magical sites, and curious landmarks
  • Resources: Food, water, shelter, and peculiar discoveries

Controls

  • Generate Map: Create a new random hex map
  • Click Hexes: View detailed information about each location
  • Regenerate: Create a completely new map with different seed
  • Export: Save your map for use in sessions

Hex Map System

Each hex represents approximately one day's travel (6 miles). The map uses a point-topped hexagonal grid for easy navigation. Terrain features influence travel time, encounter likelihood, and available resources.

Mythic Bastionland Setting

The Realms are a strange and wonderful place beyond the city of Bastionland. Ancient ruins dot the landscape, mythic beasts roam the wilds, and reality itself bends in unexpected ways. This generator captures that sense of wonder and weirdness.
